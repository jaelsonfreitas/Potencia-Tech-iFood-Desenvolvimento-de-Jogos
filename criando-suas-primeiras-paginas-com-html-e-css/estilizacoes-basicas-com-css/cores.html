Cores

Existem várias formas de definirmos cores utilizando o CSS.

São elas:

  • Cores pré-definidas
  • Palavra-chave "currentcolor"
  • Hexadecimal
  • Hexadecimal com tranparência
  • RGB
  • RGBA
  • HSL
  • HSLA

HSL

hsl (hue, saturation, lightess)
  • Hue (matriz): é o grau na roda de cores (de 0 a 360 graus)
    • 0 á 360º é vermelho.
    • 120º é o verde
    • 240º é azul
  • Saturation (saturação): Valor percentual.
    • 0 % um tom de cinza
    • 100 % cor total
  • Lightness (luminosidade): luminosidade da cor.
    • 0 % é preto.
    • 100 % é branco

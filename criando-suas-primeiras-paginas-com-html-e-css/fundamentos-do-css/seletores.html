No CSS nós possuimos os seletores por ID

  • Seletor por Tipo - Busca elementos por uma tag HTML especifica
  • Seletor por ID (#) - Busca elementos atraves do atributo "id"
  • Seletor por Classe (.) - Busca elementos através do atributo "class"
  • Seletor Universal (*) - Seleciona todos os elementos HTML.
  • Seletor de Atributo([atrib]) - Seleciona elementos que possuem atributos especificos em nosso documento HTML, e conseguimos também buscar atributos com um valor especifico.

Agrupamento de seletores

Podemos aplicar as mesmas regras CSS para vários seletores diferentes. Para isso, podemos separar seletores utilizando uma vírgula (,).

Assim não precisamos ficar copiando e colando os mesmos códigos para todos, nós podemos fazer um só bloco de CSS e reutilizarmos as mesmas regras!
